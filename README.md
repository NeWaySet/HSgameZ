# Qt Battlegrounds

Прототип авто-баттлера в духе `Hearthstone: Battlegrounds` на `C++17 + Qt`.

Что реализовано:

- герой с запасом здоровья;
- фаза покупки с монетами, стартующими с `3` и растущими каждый раунд;
- таверна на `7` существ с обновлением за `1` монету;
- рука и стол до `7` существ;
- покупка, выставление на стол и продажа существ;
- авто-бой, где существа атакуют сами по случайной цели;
- режим `сам с собой` на одном компьютере с переключением между двумя игроками;
- скрытие руки, таверны и текущего стола при передаче хода в режиме `сам с собой`;
- локальная сеть через `TCP`;
- опциональный `Bluetooth`, если модуль `Qt Bluetooth` установлен в вашей сборке Qt.

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Игровой цикл

1. Подключите двух игроков по `LAN` или `Bluetooth`.
2. Каждый игрок покупает существ в таверне.
3. Купленные карты попадают в руку.
4. Из руки существа выставляются на стол.
5. Таверну можно обновить за `1` монету.
6. После окончания покупок нажмите `Готов к бою`.
7. Когда готовы оба игрока, хост рассчитывает авто-бой и рассылает результат.

## Локальный режим

1. В поле режима выберите `Сам с собой`.
2. Нажмите `Начать локальную игру`.
3. Соберите стол для текущего игрока.
4. Нажмите `Готов к бою`, чтобы передать ход второй стороне.
5. Экран скрывается, и следующий игрок открывает свой ход кнопкой `Показать ход`.
6. При необходимости используйте `Передать управление`, чтобы вручную переключаться между игроками.
7. Когда обе стороны готовы, бой рассчитается локально без сети.

## Примечания

- урон герою проигравшего равен количеству выживших существ победителя;
- в текущем прототипе поле после боя не получает постоянных ранений, как в автобаттлерах;
- `Bluetooth` появляется в интерфейсе только если проект собран с модулем `Qt Bluetooth`.
